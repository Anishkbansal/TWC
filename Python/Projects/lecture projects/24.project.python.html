Create a personal finance manager that tracks income and expenses, categorizes transactions, and generates reports. The program will read and write data to a file, handle various types of transactions, and allow the user to query their financial data.

Steps:

Define Data Structure:
Use JSON to store and retrieve data for transactions.
Each transaction should have attributes such as date, amount, category, and description.
Create Functions for Transactions:

Add income or expense.
Categorize transactions (e.g., Food, Rent, Utilities, etc.).
Generate a report of expenses and income.

File I/O Operations:
Read and write transactions to a JSON file.
Ensure data integrity and handle file-related exceptions.

Exception Handling:
Handle invalid inputs and custom errors for financial transactions.

1) create functions for each phase for better readability
2) user should be able to add a transaction by providing necessary details.
3) user should be able to generate report about total income and total expenses 
4) add custom error messages in case user provide invalid information and do proper exception handeling
