Peepo: 
Explain the concept of inheritance in Python with an example.

Einstein: 
Inheritance allows a class to inherit attributes and methods from another class. This helps in reusability and organization of code.

class Animal:
    def __init__(self, name):
        self.name = name
    
    def speak(self):
        raise NotImplementedError("Subclass must implement abstract method")

class Dog(Animal):
    def speak(self):
        return f"{self.name} says Woof!"

class Cat(Animal):
    def speak(self):
        return f"{self.name} says Meow!"

# Example usage:
dog = Dog("Buddy")
cat = Cat("Whiskers")
print(dog.speak())  # Output: Buddy says Woof!
print(cat.speak())  # Output: Whiskers says Meow!

Could you explain the difference between deep copy and shallow copy in Python?(Opps forgot to teach that during lectures)

Einstein: 
A shallow copy creates a new object but inserts references into it to the objects found in the original. A deep copy creates a new object and recursively copies all objects found in the original, making new instances of everything. which means if you change original list after shallow copying, it will also change the copied list as they both refrence same list.(different names of same person)

import copy

original_list = [[1, 2, 3], [4, 5, 6]]
shallow_copy = copy.copy(original_list)
deep_copy = copy.deepcopy(original_list)

original_list[0][0] = 'Changed'

print(shallow_copy)  # Output: [['Changed', 2, 3], [4, 5, 6]]
print(deep_copy)     # Output: [[1, 2, 3], [4, 5, 6]]
