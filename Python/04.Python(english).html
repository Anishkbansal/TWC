Logical Operators

Peepo: 
Logical operators are similar to Comparison operators. Lets list them down first


1. 'in' membership operator. True if value exists in the sequence(List, Tuple, String etc)
2. 'not in'. True is value does not exits in the sequence
3. 'is'. True if both the values refer to same container/object (like == which also compare address of the value)
4. 'is not'. True of both values does not refer to same container/object (like != which also compares the address of the value)



            

print('a' in 'apple')  
# True if 'a' exists in string 'apple'

            print('a' not in 'apple')
            
# True if 'a' does not exists in string 'apple'

            print('z' not in 'apple')
            
# True if 'z' does not exists in string 'apple'
