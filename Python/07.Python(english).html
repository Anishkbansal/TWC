we got the break statement now, what continue statement do?
continue statement forces the loop to stop everything it is doing ad move on to the next iteration.
suppose you have 5 lines of code inside a loop and there is a continue statement at the 2nd line. so when python reaches line 2, it will stop everything and moves on to next iteration of the loop which means that the lines after continue staement will not be executed
