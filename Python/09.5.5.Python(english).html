sets are simple thing. it is like list where no item can repeat iselft like list =[0,0,0,5,4,0]  set = {0,5,4}  each item is unique in a set

Peepo: dont ever talk to me like that again. you will regret your every decision.  yes there is some difference. lets disscus it first.


1. sets are mutable .
2. set does not have any sequence like dictionary
3. no item can be repeated. no duplicates. unique characters.
4. represented using curly braces.
5. mathematic operations like intersection, union etc can be used
6. set() function can be used to convert list, tuple etc to set
7. usually used to eliminate duplicate item from a data.
8. Membership Testing: Check for membership in a collection efficiently.
